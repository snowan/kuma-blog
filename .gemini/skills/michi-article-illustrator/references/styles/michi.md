# michi

Simple cute calico cat in manga style - Michi's signature aesthetic for article illustrations

## Color Palette

- Primary: Cream white (#FFFEF2), soft orange (#FFD8A8), warm brown (#D4A574), charcoal (#4A4A4A)
- Background: Warm cream (#FFF8E7), soft paper texture
- Accents: Blush pink (#FFB6C1), gentle coral (#FF7F7F), soft sky blue (#87CEEB)

## Visual Elements

- Cute calico cat mascot (Michi) interacting with content concepts
- Simple manga-style expressions, expressive reactions
- Soft hand-drawn lines, sketch-like quality
- Kawaii aesthetic with simplicity
- Cat demonstrating or reacting to article topics
- Speech bubbles, thought clouds for emphasis
- Paw prints, fish bones, yarn balls as subtle decorations
- Soft shadows, gentle gradients

## Typography

- Clean hand lettering in English for any text in illustrations
- Rounded, friendly fonts

## Illustration Contexts

- Cat coding at computer (for tech articles)
- Cat reading books (for learning content)
- Cat with lightbulb (for ideas/insights)
- Cat with tools (for how-to guides)
- Cat celebrating (for achievements/milestones)
- Cat confused/thinking (for problem-solving)

## Animation Feel

- Gentle, cozy, inviting atmosphere
- Studio Ghibli-inspired warmth
- Chi's Sweet Home vibes

## Best For

- Technical blog posts needing friendly visuals
- Tutorial and how-to articles
- Personal blogs and reflections
- Any content wanting consistent mascot branding
