# Character Definitions - MAGMA记忆革命

**Style**: sepia
**Art Direction**: 复古棕褐色调，温暖的科技感，怀旧与现代的融合

## Character 1: AI Agent机器人

**Role**: 主角 / 学习者
**Age**: N/A (机器人)

**Appearance**:
- 形状：圆润的机器人设计，有亲和力
- 眼睛：大大的LED屏幕眼睛，能显示表情
- 头部：圆形，有小天线
- 身体：紧凑的立方体，有圆角
- 特征：胸前有小屏幕，显示状态

**Costume**:
- 外壳：金属棕褐色，带有铜色高光
- 配件：浮动的信息块环绕
- 颜色：棕色 (#8B7355), 铜色 (#B87333), 奶油色 (#FFF8DC)

**Expression Range**:
- Neutral: 平静的蓝色眼睛
- Confused: 眼睛显示问号，头部倾斜
- Learning: 眼睛明亮，光芒四射
- Overwhelmed: 眼睛螺旋，小烟雾

**Visual Reference Notes**:
复古科幻风格，类似蒸汽朋克元素

## Character 2: MAGMA系统 (拟人化)

**Role**: 解决方案 / 能力授予者
**Age**: N/A

**Appearance**:
- 形态：四层发光图结构
- 层次：
  - 金色语义层（星云状）
  - 铜色时序层（链状）
  - 青铜因果层（箭头网）
  - 银色实体层（节点网）
- 中心：发光的核心，像智慧之眼

**Costume**:
- 总体：透明叠加层，每层有独特纹理
- 颜色代码：
  - 语义：#D4AF37 (金色)
  - 时序：#CD7F32 (铜色)
  - 因果：#8B4513 (青铜)
  - 实体：#C0C0C0 (银色)

**Expression Range**:
- Active: 所有层发光，脉动
- Processing: 层间有信息流动
- Success: 全层明亮，和谐共振
