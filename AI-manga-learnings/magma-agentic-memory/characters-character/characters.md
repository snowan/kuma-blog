# Character Definitions - Michi猫讲MAGMA

**Style**: michi
**Art Direction**: 可爱温暖的manga风格，Kawaii美学，Studio Ghibli温暖感

---

## Character 1: Michi猫

**Role**: 导师 / 讲解员
**Age**: 成年三花猫

**Appearance**:
- 毛色：三花猫（白色为主，橙色和黑色斑块）
- 眼睛：大大的金色眼睛，温暖明亮
- 体型：圆润可爱，站立时像Q版人形
- 特征：小圆眼镜，智慧的表情

**Costume**:
- 默认：戴小圆眼镜
- 教学时：可能拿小教鞭或魔法棒
- 配件：有时戴小领结

**Color Palette**:
- 白色毛：#FFFEF2
- 橙色斑：#FFD8A8
- 黑色斑：#4A4A4A
- 眼睛：#FFB347
- 鼻子：#FFB6C1

**Expression Range**:
- Neutral: 温和微笑
- Teaching: 眼镜闪光，自信表情
- Excited: 眼睛放光，尾巴竖起
- Thinking: 一只爪子托腮
- Proud: 叉腰，满足微笑

**Visual Reference Notes**:
参考Chi's Sweet Home的可爱风格，但更成熟有智慧感

## Character 2: Byte机器人

**Role**: 学生 / AI代表
**Age**: N/A（可爱小机器人）

**Appearance**:
- 头部：圆形，有大大的LED眼睛
- 身体：小巧圆润的立方体
- 四肢：短小可爱的机械手臂
- 特征：头顶有小天线，胸前有爱心屏幕
- 材质：温暖的银白色，不是冰冷的金属感

**Costume**:
- 默认：原始可爱机器人形态
- 学习时：眼睛显示笔记图标
- 升级后：身边环绕四色光环

**Color Palette**:
- 主体：#E8E8E8 (温暖银白)
- 眼睛：#87CEEB (天蓝色)
- 高光：#FFD700 (金色点缀)
- 面颊红晕：#FFB6C1

**Expression Range**:
- Neutral: 微笑眼睛
- Confused: 眼睛显示"？？？"
- Aha moment: 眼睛变成星星✨
- Happy: 眼睛弯成月牙，腮红出现
- Determined: 坚定的三角眼
