# Character Definitions - MAGMA多维记忆

**Style**: tech
**Art Direction**: 科技蓝调，霓虹高光，干净线条，数字感

---

## Character 1: 四图层系统

**Role**: 主角 / 架构可视化
**Type**: 抽象系统图形

**Appearance**:
- 形态：四个嵌套的立体结构
- 语义层：蓝色立方体，内有节点云
- 时序层：绿色圆柱，内有时间线
- 因果层：橙色金字塔，内有箭头
- 实体层：紫色球体，内有连接网

**Color Palette**:
- 语义：#00CED1 (青色)
- 时序：#32CD32 (绿色)
- 因果：#FF8C00 (橙色)
- 实体：#9370DB (紫色)
- 背景：#0D1B2A (深蓝)

**Visual States**:
- Static: 清晰分层展示
- Active: 层间有数据流动
- Query: 特定层高亮

---

## Character 2: Query Router

**Role**: 引导 / 分发器
**Type**: 功能组件

**Appearance**:
- 形态：中央控制台形象
- 三个输出端口：When/Why/Who
- 指示灯显示当前路由
- 数据流从输入到输出

**Color Palette**:
- 主体：#2F4F4F (深青灰)
- 端口：对应图层颜色
- 活跃：霓虹高光

---

## Character 3: Performance Metrics

**Role**: 验证 / 结果展示
**Type**: 数据可视化元素

**Appearance**:
- 柱状图形态
- MAGMA柱最高（金色）
- 竞争对手柱较矮（灰色）
- 数字标签清晰
