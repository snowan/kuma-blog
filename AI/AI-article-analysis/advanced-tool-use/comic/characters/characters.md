# Character Reference: Michi

## Main Character: Michi the Agent Cat

**Species**: Calico cat (三毛猫)
**Role**: AI Agent learning to use tools effectively

### Visual Appearance
- **Fur Pattern**: Cream base with orange and gray/black patches
- **Eyes**: Large, expressive, amber/golden
- **Accessory**: Round glasses (slightly nerdy, but cute)
- **Extra**: Small blue ribbon bow on collar
- **Style**: Japanese manga kawaii aesthetic

### Expression Range
1. **Overwhelmed**: Spiral eyes, sweat drops, papers flying
2. **Curious**: Head tilt, one ear up, sparkle in eye
3. **Thinking**: Paw on chin, looking up
4. **Triumphant**: Confident smile, sparkles around, victory pose
5. **Lightbulb Moment**: Eyes wide, exclamation mark above head

### Poses
- Sitting at desk with tablet/laptop
- Buried under pile of papers
- Holding magnifying glass (searching)
- Writing code (focused look)
- Juggling items (multitasking)
- Reading a book (learning examples)

## Supporting Visual Elements

### The Tool Mountain
- Stack of 100+ scrolls/folders
- Labels peeking out: "GitHub", "Slack", "Jira"
- Overwhelming, chaotic energy

### The Three Powers (Icons)
1. **Tool Search**: Glowing magnifying glass
2. **PTC (Code)**: Python code block with gears
3. **Examples**: Notebook with checkmark badges

### Stats/Metrics Badges
- Token counts in pill-shaped bubbles
- Accuracy percentages with arrows (49→74%)
- Modern, clean design
