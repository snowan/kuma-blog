# Character Definitions - Building MAGMA: Layer by Layer

**Style**: dramatic
**Art Direction**: High contrast, cinematic lighting, architectural precision, deep shadows

---

## Character 1: Data Structure Layer

**Role**: Foundation - the storage substrate
**Form**: Architectural foundation structure

**Appearance**:
- Structure: Solid hexagonal platform with depth
- Color: Deep blue (#1A365D) with blue glow (#3182CE)
- Surface: Grid patterns, crystal formations
- Contains: Vector database icon, four graph structures embedded
- Visual weight: Heavy, solid, dependable

**Visual Identity**:
- Icon: Layered database symbol with graph overlay
- Glow effect: Upward blue illumination
- Lighting: Lit from below, casting upward shadows

---

## Character 2: Query Process Layer

**Role**: Brain - the processing engine
**Form**: Active processing platform

**Appearance**:
- Structure: Hexagonal platform with visible internal machinery
- Color: Emerald green (#00FF88) as primary glow
- Surface: Active processing indicators, flowing data paths
- Contains: Router component, retrieval engine, synthesizer
- Visual weight: Dynamic, active, intelligent

**Visual Identity**:
- Icon: Brain with query arrows
- Glow effect: Pulsing green energy
- Lighting: Internal glow with external radiation

---

## Character 3: Write/Update Layer

**Role**: Evolution - memory growth and refinement
**Form**: Dual-stream processing platform

**Appearance**:
- Structure: Hexagonal platform split into two channels
- Color: Orange (#FFA500) for fast path, Amber (#FFB347) for slow path
- Surface: Left side is quick/lightning, right side is contemplative
- Contains: Ingestion queue, consolidation workers
- Visual weight: Balanced between speed and depth

**Visual Identity**:
- Icon: Lightning bolt (fast) + Hourglass (slow)
- Glow effect: Dynamic orange on left, warm amber on right
- Lighting: Dramatic side lighting emphasizing duality

---

## Character 4: Query (Visual Representation)

**Role**: Input trigger
**Form**: Glowing data packet

**Appearance**:
- Structure: Crystalline shard or orb
- Color: White with cyan edge glow
- Surface: Smooth, containing text
- Movement: Descending from above, entering the system
- Size: Small relative to layers

**Visual Identity**:
- Icon: Question mark in glowing bubble
- Glow effect: Soft white aura
- Lighting: Self-illuminated

---

## Character 5: Intent Router

**Role**: Query classifier within Query Layer
**Form**: Central processing node

**Appearance**:
- Structure: Octagonal node with three output channels
- Color: Green core (#00FF88)
- Outputs: "Why?" (orange), "When?" (green), "Entity?" (purple)
- Animation: Scanning effect, classification visual

**Visual Identity**:
- Icon: Branching arrow from center
- Glow effect: Scanning beam
- Lighting: Central spotlight

---

## Character 6: Retrieved Subgraph

**Role**: Query result
**Form**: Assembled collection of nodes

**Appearance**:
- Structure: Mini-graph extracted from main structure
- Color: Multi-colored based on contributing edges
- Surface: Connected, coherent, organized
- Movement: Floating upward as result
- Size: Variable based on query complexity

**Visual Identity**:
- Icon: Connected nodes in bubble
- Glow effect: Harmonious multi-color
- Lighting: Unified glow indicating synthesis
