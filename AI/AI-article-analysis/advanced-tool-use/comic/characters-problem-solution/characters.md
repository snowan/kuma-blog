# Character Definitions - MAGMA: The Memory Revolution

**Style**: tech
**Art Direction**: Clean geometric lines, neon highlights, digital/cyber aesthetic

---

## Character 1: The AI Agent (MAX)

**Role**: Protagonist - represents LLM/AI systems
**Age**: N/A (anthropomorphized AI)

**Appearance**:
- Face shape: Sleek, geometric, humanoid robot
- Hair: None - smooth metallic head with LED patterns
- Eyes: Dual glowing cyan circles, expressive
- Build: Tall, lean, elegant robotic form
- Distinguishing features: Glowing circuit patterns on body, core light in chest

**Costume**:
- Default outfit: Sleek white/silver robotic exterior with black accents
- Color palette: White (#FFFFFF), silver (#C0C0C0), cyan glow (#00D4FF)
- Accessories: Holographic interface displays around hands

**Expression Range**:
- Neutral: Steady cyan glow, balanced posture
- Confused/Struggling: Flickering lights, tilted head, dimmed glow
- Understanding/Success: Bright cyan pulse, upright posture
- Processing: Spinning ring indicators around head

## Character 2: Semantic Graph (SEMA)

**Role**: Supporting - represents semantic relationships
**Form**: Abstract network visualization

**Appearance**:
- Structure: Clustered constellation of nodes
- Color: Cyan blue (#00D4FF)
- Nodes: Soft rounded shapes, varying sizes
- Edges: Undirected, flowing connections
- Movement: Gentle pulsing, like breathing

**Visual Identity**:
- Icon: Three overlapping circles (like a Venn diagram)
- Glow effect: Soft diffuse cyan aura

## Character 3: Temporal Graph (TEMPO)

**Role**: Supporting - represents time relationships
**Form**: Timeline chain structure

**Appearance**:
- Structure: Linear chain of connected nodes
- Color: Emerald green (#00FF88)
- Nodes: Diamond shapes marking time points
- Edges: Directed, flowing arrows along time
- Movement: Flowing light moving in one direction

**Visual Identity**:
- Icon: Horizontal arrow with clock
- Glow effect: Flowing green energy like timeline stream

**Form**: Chaotic mess of connections

- Structure: Tangled spaghetti of nodes and edges
